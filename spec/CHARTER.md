# Charter

## What this benchmark measures

This benchmark measures the practical usefulness of coding systems as they are actually used:

- model + native harness;
- on shared tasks;
- with shared repo instructions;
- judged by the same human reviewer.

## What this benchmark does not try to do

- It does not isolate the bare model from its harness.
- It does not produce an objective universal ranking.
- It does not replace human judgment with a formula.

## Benchmark outputs

The repository should support three outputs:

- a raw run archive;
- a human review sheet per task;
- a final winner decision per benchmark wave.
