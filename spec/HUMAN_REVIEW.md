# Human Review

The primary reviewer is the user.

## Main principle

The benchmark winner is chosen by human preference after seeing produced results.

## Default review axes

- works;
- looks good;
- feels finished;
- follows the task;
- shows restraint;
- would I actually want to use it?

## Output form

Each task review should end with:

- short score or label per model;
- one-line comment per model;
- explicit winner for that task.
