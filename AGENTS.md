# AGENTS.md

Shared benchmark instructions copied into every task workspace.

## Core rule

Solve the task exactly as written.
Do not rewrite the task into a different project.

## Constraints

- Keep changes local to the provided workspace.
- Do not fetch external dependencies unless the task explicitly allows it.
- Produce the exact submission artifact required by the task pack.
- Stop when the task is complete; do not add extra features.

## Output discipline

- If the task requires a file, create the file.
- If the task requires a runnable app, leave it runnable in place.
- If the task requires a visual artifact, make it easy to inspect.
